🥄 Receitas Colaborativas

Partilha, aprende e colabora com os teus colegas num ambiente inspirador!

Como Colaborar

  1. Saca o repositório usando git clone.
  2. Cria uma branch para cada receita ou melhoria (git checkout -b feature/nome-receita).
  3. Faz commits das alterações.
  4. Abre um Pull Request para integrar ao projeto principal.
  5. Usa Issues para sugestões e resolução de problemas.

Receitas em Desenvolvimento

  • 🍫 feature/brigadeiro - Adição da receita de brigadeiro
  • 🥘 fix/tempo-feijoada - Correcção no tempo de cozedura da feijoada
