Colabore em Receitas de Culinária

Aprenda a usar Branches, Pull Requests e Issues no GitHub com receitas!

Fluxo de Trabalho Colaborativo

  1. Crie branches para cada receita ou tarefa (ex: feature/bolo-cenoura).
  2. Faça commits com mensagens claras e informativas.
  3. Crie Pull Requests e solicite revisão dos colegas.
  4. Utilize Issues para gerenciar tarefas e ideias.
