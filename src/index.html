Receitas Colaborativas.

Partilha e aprende a utilizar GitHub com receitas!

Como colaborar

  1. Saca o repositório usando git clone.
  2. Cria uma branch para cada receita nova ou correção (git checkout -b feature/nome-receita).
  3. Faz commits das tuas alterações.
  4. Abre um Pull Request para integrar no projeto principal.
  5. Utiliza Issues para propor melhorias ou assinalar problemas.

Receitas em desenvolvimento

  • feature/brigadeiro (Adição da receita de brigadeiro)
  • fix/tempo-feijoada (Correcção no tempo de cozedura da feijoada)
